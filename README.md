# HR-Analytics-Dashboard

## **HR Analytics: Employee Attrition & Workforce Insights Dashboard**

An interactive Power BI dashboard designed to analyze employee attrition and workforce patterns, helping organizations improve employee performance, employee experience, and retention through data-driven insights.

## Purpose
This dashboard analyzes HR data to identify the key factors influencing employee attrition and performance.  
It supports HR teams in making informed decisions to reduce attrition and build a more productive workforce.

## Tech Stack
The dashboard was built using the following tools and technologies:

- **Power BI Desktop** – Primary tool for data visualization and dashboard creation  
- **Power Query** – Data cleaning, transformation, and preprocessing  
- **DAX (Data Analysis Expressions)** – Calculated measures and KPIs  
- **Excel / CSV** – Source data format  

## Data Source
**Source:** HR Analytics Dataset (Kaggle)

The dataset contains employee-level information including age, gender, department, job role, salary, education, years at company, and attrition status.  
This data was used to derive meaningful insights related to employee retention and performance.

## Features / Highlights

### Business Problem
Employee attrition negatively impacts organizational productivity, hiring costs, and employee morale.  
Many organizations lack a clear and visual understanding of the factors contributing to attrition.

### Goal of the Dashboard
- Identify key drivers of employee attrition  
- Improve employee performance and experience  
- Support better hiring and retention strategies  
- Enable data-driven HR decision-making  

### Walkthrough of Key Visuals
- **Attrition Overview KPIs** – Total employees, attrition count, and attrition rate  
- **Attrition by Age Group** – Identifies age ranges with higher turnover  
- **Attrition by Education** – Highlights education-wise attrition trends  
- **Attrition by Salary Slab** – Shows higher attrition at lower salary levels  
- **Attrition by Job Role** – Compares attrition across different roles  
- **Attrition by Years at Company** – Reveals tenure-based exit patterns  
- **Department Filters** – HR, Research & Development, and Sales  

### Business Impact & Insights
- Identified key factors contributing to employee attrition  
- Improved hiring and workforce planning decisions  
- Enhanced employee experience and engagement  
- Supported strategies to reduce attrition and improve retention  
- Helped build a more productive workforce  

## Data Cleaning & Processing
- Removed duplicate records  
- Handled missing and inconsistent values  
- Created age groups and salary slabs  
- Transformed and processed data using Power Query

## Conclusion
This HR Analytics Dashboard demonstrates how data visualization can support strategic HR decisions.  
By identifying attrition patterns and workforce trends, the dashboard helps organizations take proactive actions to improve employee retention, enhance performance, and optimize workforce productivity.  
It highlights the importance of data-driven approaches in modern human resource management.

## How to Use
1. Download the `HR_Analytics_Dashboard.pbix` file  
2. Open it using **Power BI Desktop**  
3. Interact with filters and visuals to explore insights
